# Monitoring-Water-Level
Monitoring Sms Gateway SIM900A, Sensor Ultrasonic HC-04

# Hardware
- Arduino Uno / Nano
- Module Relay 1 CH
- SIM900A
- Ultrasonic HC-04
- Solenoid Valve
- PSU 5V 5A
